# Fullstack Project – Next.js (Frontend) + NestJS (Backend)

Este proyecto combina:

* **Next.js** para el frontend
* **NestJS** para el backend

## 🚀 Tecnologías principales

### Frontend (Client)

* Next.js
* TypeScript
* next/font
* App Router

### Backend (Server)

* NestJS
* Node.js
* TypeScript

---

# 📁 Frontend – Next.js

Este es un proyecto creado con [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## ▶️ Iniciar el proyecto

Ejecuta el servidor de desarrollo:

```bash
npm run dev
# o
yarn dev
# o
pnpm dev
# o
bun dev
```

Luego abre:

```
http://localhost:3000
```

La página se puede editar modificando:

```
app/page.tsx
```

El proyecto usa [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) para optimizar la carga de tipografías como **Geist**.

## 📚 Recursos recomendados

* Documentación oficial: [https://nextjs.org/docs](https://nextjs.org/docs)
* Tutorial interactivo: [https://nextjs.org/learn](https://nextjs.org/learn)
* Repositorio oficial: [https://github.com/vercel/next.js](https://github.com/vercel/next.js)

## ☁️ Deploy en Vercel

El método más sencillo es desplegar con:

Consulta la documentación de despliegue:

[https://nextjs.org/docs/app/building-your-application/deploying](https://nextjs.org/docs/app/building-your-application/deploying)

# 🖥 Backend – NestJS

<p align="center">
  <a href="http://nestjs.com/" target="blank">
    <img src="https://nestjs.com/img/logo-small.svg" width="120" alt="Nest Logo" />
  </a>
</p>

NestJS es un framework progresivo para crear aplicaciones del lado del servidor, eficientes y escalables sobre Node.js y TypeScript.

## 📦 Instalación

```bash
npm install
```

## ▶️ Ejecutar el servidor

```bash
# desarrollo
npm run start

# watch mode
npm run start:dev

# producción
npm run start:prod
```

## 🧪 Tests

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# cobertura
npm run test:cov
```

## ☁️ Deploy

Guía oficial de despliegue:

[https://docs.nestjs.com/deployment](https://docs.nestjs.com/deployment)

Para deploy en AWS usando **Mau**:

```bash
npm install -g @nestjs/mau
mau deploy
```
